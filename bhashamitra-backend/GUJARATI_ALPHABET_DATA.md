## Notes
- Gujarati uses the same example words as Hindi where applicable (e.g., અનાર/अनार for pomegranate)
- Some letters don't have common beginner-friendly example words and are marked with "-"
- Images are sourced from Unsplash to match the existing pattern in the frontend
- The connector word in Gujarati for "for example" is "થી" (thi) or just use the pattern "letter - word"
