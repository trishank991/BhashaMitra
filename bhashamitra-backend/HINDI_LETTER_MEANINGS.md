## Notes

- All example words follow the traditional Hindi learning pattern: "X से Y" (X for Y)
- For example: "अ से अनार" means "A for Pomegranate"
- Images are sourced from Unsplash and match the meaning of each example word
- This ensures visual learning reinforcement for children learning the Hindi alphabet
