# Peppi Academy - Platform Architecture

## Vision Statement
**Peppi Academy** is an inclusive, comprehensive online learning platform celebrating India's incredible diversity. We serve diaspora children and young adults worldwide, helping them connect with their heritage across languages, mathematics, history, and culture - representing ALL of India's religions, regions, and traditions.

### 2. Peppi Maths (Vedic Mathematics & Beyond)
**Tagline**: "Ancient Wisdom, Modern Learning"

#### Curriculum
| Level | Age | Topics |
|-------|-----|--------|
| **Foundation** | 4-6 | Number recognition, counting, basic patterns |
| **Primary** | 7-9 | Vedic sutras for addition, subtraction, multiplication |
| **Intermediate** | 10-12 | Mental math, quick calculation techniques |
| **Advanced** | 13-16 | Complex calculations, competitive math prep |

#### Vedic Mathematics Topics
1. **Ekadhikena Purvena** - Squaring numbers ending in 5
2. **Nikhilam** - Multiplication near base 10/100/1000
3. **Urdhva Tiryagbhyam** - Crosswise multiplication
4. **Paravartya Yojayet** - Division techniques
5. **Shunyam Saamyasamuccaye** - Zero and equality
6. **Anurupye** - Proportionality
7. **Sankalana-Vyavakalanabhyam** - Addition/subtraction method
8. **Puranapuranabhyam** - Completion techniques

#### Beyond Vedic Maths
- **Aryabhata's contributions** - Place value system
- **Brahmagupta's mathematics** - Zero as a number
- **Indian mathematicians** - Ramanujan, Bose, Rao
- **Logic puzzles** - Brain teasers with Indian themes

### Database Models (New)

#### Maths Module
```python
class MathTopic(TimeStampedModel):
    name = models.CharField(max_length=200)
    category = models.CharField(choices=['VEDIC', 'MENTAL', 'LOGIC', 'COMPETITION'])
    difficulty = models.IntegerField(1-5)
    prerequisite = models.ForeignKey('self')

class MathLesson(TimeStampedModel):
    topic = models.ForeignKey(MathTopic)
    title = models.CharField(max_length=200)
    explanation = models.TextField()
    examples = models.JSONField()
    practice_problems = models.JSONField()

class MathProgress(TimeStampedModel):
    child = models.ForeignKey(Child)
    topic = models.ForeignKey(MathTopic)
    problems_attempted = models.IntegerField()
    problems_correct = models.IntegerField()
    mastered = models.BooleanField()
```

#### History Module
```python
class HistoricalEra(TimeStampedModel):
    name = models.CharField(max_length=200)
    start_year = models.IntegerField()
    end_year = models.IntegerField()

class HistoricalTopic(TimeStampedModel):
    era = models.ForeignKey(HistoricalEra)
    title = models.CharField(max_length=200)
    description = models.TextField()
    key_figures = models.JSONField()
    key_events = models.JSONField()

class HistoryStory(TimeStampedModel):
    topic = models.ForeignKey(HistoricalTopic)
    title = models.CharField(max_length=200)
    content = models.TextField()
    age_range = models.CharField(choices=['4-6', '7-9', '10-12', '13+'])
    includes_quiz = models.BooleanField()
```

#### Culture Module
```python
class Festival(TimeStampedModel):

    origin_region = models.CharField()
```
